Introduction

The PH4502C sensor is a high-precision pH sensor capable of accurately measuring pH levels in various applications. To use this sensor you can install Arduino IDE library or calibrate it manually.

Features

This sensor has features such as:

  • Supports pH measurement with high precision and accuracy.
  • Flexible calibration options to ensure optimal performance.
  • Supports 3.3V and 5V voltage as input for Arduino/ESP board.
Board Description

This board description is describing about PH4502C board as followed:

  • BNC Connecter: Probe connector
  • To: Temperature Pin
  • Do: Adjustable Voltage Pin
  • Po: Input Pin
  • GND: Ground Pin
  • VCC: Power Input Pin
  • Blue Potensiometer: Adjustable pH offset value

Installation on Arduino IDE

To use the PH4502C Sensor Arduino Library, follow this steps:

  1. Open Arduino IDE, if you don't have Arduino IDE, download here
  2. Open Library Manager of Arduino IDE, then type "PH4502C"
  3. Install the Library, It's an open-source library by nthnn

Getting Started

For started to used PH4502C Sensor, follow this documentation below:

  1. Connect your PH4502C Sensor to your board accordingly based on pin configuration.
  2. In your C++ sketch, import the PH4502C Sensor library by adding these line below:
    3. #include ph4502c_sensor.h

    Create an instance of the PH4502C class:

    PH4502C ph4502c(A0,A1);

    A0 is for pH input and A1 is for Temperature input

  3. Initialize the sensor in the void setup() function by using:
    4. ph4502c.init();
  4. Read the pH value using the function inside void loop()
    5. ph4502c.read_ph_level();

    Make sure that you read the pH value from inside a variables, such as below:

    float pH_value = ph4502c.read_ph_level();

    Alternatively, you can calibrate the sensor manually. Refer to this medium article for futher information.
